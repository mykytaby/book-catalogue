# 📚 Каталог книг

Настільний застосунок на Java (Swing) для керування каталогом книг.  
Лабораторна робота · П'ятаченко В.Ю.

---

## Можливості

- Додавання, редагування та видалення книг
- Пошук у реальному часі (за назвою та автором)
- Сортування за назвою або роком видання
- Збереження та завантаження каталогу з файлу (серіалізація)
- Сучасний темний інтерфейс із таблицею записів

## Запуск через Maven (рекомендовано)

### 1. Клонувати репозиторій

```bash
git clone https://github.com/<ВАШ_НІКНЕЙМ>/book-catalogue.git
cd book-catalogue
```

### 2. Зібрати проєкт

```bash
mvn package
```

Maven скомпілює код і створить готовий файл у папці `target/`:

```
target/book-catalogue-1.0.0-runnable.jar
```

### 3. Запустити

```bash
java -jar target/book-catalogue-1.0.0-runnable.jar
```

---

## Запуск вручну (без Maven)

Якщо Maven не встановлено, можна запустити вручну через `javac` / `java`.

### 1. Перейти до папки з вихідними файлами

```bash
cd src/main/java
```

### 2. Скомпілювати всі класи

```bash
javac -encoding UTF-8 *.java
```

### 3. Запустити головний клас

```bash
java BookGUI
```

> **Примітка для Windows.** Якщо українські символи відображаються некоректно, додайте прапорець кодування:
> ```bash
> java -Dfile.encoding=UTF-8 BookGUI
> ```

## Як користуватись програмою

1. Заповніть поля **Назва**, **Автор**, **Рік** (обов'язкові) та натисніть **＋ Додати книгу**.
2. Щоб редагувати запис — клікніть на рядок у таблиці (поля заповняться автоматично), змініть потрібні дані, натисніть **✎ Оновити книгу**.
3. Для видалення — вкажіть назву у полі або клікніть рядок, натисніть **✕ Видалити книгу** та підтвердьте.
4. Пошук працює в реальному часі — просто почніть вводити у поле пошуку.
5. **💾 Зберегти у файл** — зберігає каталог у `catalogue.dat` поряд із програмою.
6. **📂 Завантажити з файлу** — відновлює каталог із збереженого файлу.
